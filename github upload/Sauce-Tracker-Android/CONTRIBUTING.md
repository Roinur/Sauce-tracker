# Contributing

## Setup
1. Install Android Studio (or Android SDK + JDK 17).
2. Open this project folder.
3. Create `local.properties` with your SDK path if needed.
4. Run `./gradlew :app:assembleDebug` (Windows: `gradlew.bat :app:assembleDebug`).

## Code style
- Keep changes focused.
- Avoid committing generated files from `build/`.
- Never commit signing keys or passwords.

## Pull requests
- Include a short summary of what changed.
- Include test/build steps you ran.
- Attach screenshots for UI changes.
