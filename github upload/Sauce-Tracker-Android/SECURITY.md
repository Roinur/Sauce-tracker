# Security Policy

## Reporting a vulnerability
Please report security issues privately first and avoid posting exploit details publicly.

Include:
- Steps to reproduce
- Impact
- Affected version (for example: Sauce Tracker 1.2)

## Secrets policy
This repository must not contain:
- Keystore files (`*.jks`, `*.keystore`)
- Store/key passwords
- `local.properties` paths tied to local machines
